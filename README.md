# Student-Marks-Prediction
# 🎓 Student Marks Prediction System

## 📌 Project Overview
The **Student Marks Prediction System** is a Machine Learning based web application that predicts student marks according to study hours entered by the user.

The project uses a **Random Forest Regression** model integrated with a **Flask** web application to provide accurate real-time predictions through a simple and interactive interface.

---

# 🚀 Features
- Predict student marks based on study hours
- Machine Learning model using Random Forest Regressor
- Real-time prediction using Flask
- User-friendly interface
- Input validation
- Stores prediction history in CSV file
- Responsive web application
