# Student-Marks-Prediction
Student Marks Prediction System
📌 Project Overview

The Student Marks Prediction System is a Machine Learning based web application that predicts a student's marks according to the number of study hours entered by the user.
The project uses a trained Random Forest Regression model integrated with a Flask web application to provide real-time predictions through a simple and interactive user interface.

🚀 Features
Predicts student marks based on study hours
Machine Learning model using Random Forest Regressor
Interactive Flask web application
Input validation for realistic study hours
Stores prediction history in CSV file
Responsive and user-friendly interface
